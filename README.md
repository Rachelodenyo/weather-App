**weather App**

This is a simple weather application that allows users to search for any city and view real time weather data.

**Features**

-Search for any city
-displays temperature,humidity,wind,pressure
-Shows weather condition and icon
-Sunrise and sunset time
-Error handling for invalid input and network issues

**Languages and Technologies used**
-HTML
-CSS
-JavaScript
-OpenWeather API
